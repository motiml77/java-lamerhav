# אתר JAVA לבגרות — חבילת הפעלה מלאה

חבילה זו מכילה את **כל האתר** + **כל מה שצריך ל‑Firebase**, מוכן להעלאה.

---

## מה יש בחבילה

```
deploy/
├── index.html                 ← דף הנחיתה (אנימציה → מעבר לאתר)
├── app.html                   ← האפליקציה המלאה (מורה + תלמידות)
├── codemirror-java.min.js     ← עורך הקוד (Java, השלמה אוטומטית) — חובה!
├── firestore.rules            ← כללי אבטחה ל-Firebase
└── firebase-setup/
    ├── seed.js                ← הקמת כל הנושאים והאוספים בפקודה אחת (אופציונלי)
    ├── firestore.rules        ← עותק של הכללים
    └── README.md              ← הוראות מפורטות ל-Firebase
```

> שלושת הקבצים `index.html`, `app.html`, `codemirror-java.min.js` חייבים לשבת **באותה תיקייה**. דף הנחיתה מעביר אוטומטית ל‑`app.html`.

## הפעלה ב-3 שלבים

### 1. העלאת האתר
העלי את תוכן `deploy/` (ללא תיקיית `firebase-setup`, שהיא כלי צד) לאחסון —
Vercel / Netlify / Firebase Hosting / כל שרת סטטי. הקובץ הראשי הוא `index.html`.

החיבור ל‑Firebase כבר מוטמע בתוך `app.html` (פרויקט `exams-a93fb`) — אין מה להגדיר.

### 2. פרסום כללי האבטחה
Firebase Console → פרויקט `exams-a93fb` → **Firestore Database** → לשונית **Rules** →
הדביקי את כל תוכן `firestore.rules` → **Publish**.

זה אוכף: רק המורה (motiml77@gmail.com) כותבת תוכן וציונים; כל תלמידה רואה רק את התשובות שלה; מחוון המבחן נסתר מתלמידות.

### 3. יצירת הנושאים
שתי דרכים — בחרי אחת:

**א. הכי קל — דרך האתר (מומלץ):**
היכנסי כמורה (motiml77@gmail.com) → בדף הבית לחצי על הכפתור הירוק **"יצירת נושאי לימוד מובנים"**.
בלחיצה אחת נוצרים כל 10 הנושאים, משויכים אוטומטית לשכבה (יא/יב), עם ההסברים, הדוגמאות החיות וההדגמות האינטראקטיביות.

**ב. דרך סקריפט (לא חובה):**
ראי `firebase-setup/README.md` — מצריך מפתח שירות והרצת `node seed.js`.
